# Karma - Plataforma de Fidelización

Landing page moderna para Karma, una plataforma de fidelización de clientes tipo Ruklo/WelcomeBack.

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- Lucide React (iconos)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Características

- Diseño moderno con gradientes
- Responsive design
- Animaciones suaves
- Colores de startup modernos
- Secciones:
  - Hero con estadísticas
  - Características del producto
  - Precios
  - CTA
  - Footer
